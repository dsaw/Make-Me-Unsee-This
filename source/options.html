Make Me Unsee This: Configuration
Scan on Change

Look for names whenever page changes, not only at first load and periodically afterwards. This is good for pages like Twitter and Facebook that show a constant feed, but it's a bit more CPU intensive.
Replace Images

Attempt to replace images associated with names with blank images.
Site permissions

You can control which sites this plugin runs on. Choose "Sites Enabled" to explicity specify the sites you want Paint it Black - Counter Hate to run on. Choose "Sites Disabled" to run on all sites except those listed.

Sites Enabled

Enter a space-separate list of website hosts that you want to be included. Make Me Unsee This will run on these sites only.
Sites Disabled

Enter a space-separate list of websites that you do not want to be checked for offensive language. All other sites will be checked.
Show configuration details and blacked out names.
Enabled Actions

Check a name to black it out. Uncheck it to leave it be.
